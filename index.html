The interface generator is an artificial intelligence powered program that uses a neural network to create an entertaining interface.
Every user sees a unique one. What are you going to get? Nobody knows! How exciting!

Each interface is graded based on its performance.
The longer the user has kept the tab opened, and the smaller the code is, the better the interface has performed.
The best performing interfaces are then selected to breed the next generation.
Hope you enjoy!
